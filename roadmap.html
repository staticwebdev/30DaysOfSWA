Roadmap

Welcome to the #30DaysOfSWA - a curated journey to help you jumpstart your learning journey into Azure Static Web Apps (SWA).

  • We'll start with core concepts (Week 1)
  • See SWA in action with examples (Week 2)
  • Boost our developer workflow with dev tools (Week 3)
  • Think about end-to-end developer experiences (Week 4)

Here's a tentative plan for the topics we hope to cover in the upcoming #30Days. Don't forget to subscribe to our blog for updates throughout the journey, and beyond!
